# ホビーフォートレス Phase 2 パッケージ

このパッケージには、次フェーズで着手するための文書を入れています。

- `04_次フェーズ実装指示書_ホビーフォートレスMVP_v0.1.md`
- `05_IDEプロンプト_ホビーフォートレスMVP_v0.1.md`
- `06_改善提案メモ_現状所見と優先順位.md`

## ねらい
現状のプロジェクトでは、編纂室の基盤がすでに入っているため、
次フェーズは **ホビーフォートレスMVP** を最小構成で立ち上げるのが筋です。

## 使い方
1. まず `06_改善提案メモ_現状所見と優先順位.md` を読んで、現状判断を確認する
2. 次に `04_次フェーズ実装指示書_ホビーフォートレスMVP_v0.1.md` を IDE に渡す前の実装方針として使う
3. 実装に入るときは `05_IDEプロンプト_ホビーフォートレスMVP_v0.1.md` を投入する

## 方針
- 大改造しない
- 編纂室を土台にする
- まずは 1 趣味、3 階層、手動定義でよい
- 趣味の階層判定が「動く」ことを先に取る
- AI ブリッジ前提にしない
