# 編纂室 / ホビーフォートレス vNext+ パッケージ

このZIPには、以下が入っています。

- `01_ルーブリック評価_編纂室_ホビーフォートレス.md`
- `02_vNextプラス修正方針.md`
- `03_IDE実装プロンプト_編纂室_vNextプラス.md`

## 目的
うぜん向けに作成した以下2文書を、より実装に耐える形へ補強するための材料です。

- 統合構想書 vNext
- Phase 1 実装計画書: 編纂室 vNext

## このパッケージの使い方
1. まず `01_ルーブリック評価_編纂室_ホビーフォートレス.md` を読んで、現状の評価軸と弱点を確認する
2. 次に `02_vNextプラス修正方針.md` を読んで、何を補強するかを把握する
3. そのうえで `03_IDE実装プロンプト_編纂室_vNextプラス.md` を IDE に投入し、文書改訂を進める

## 補足
今回の主眼は、文書を単に整えることではなく、うぜんの運用特性に合わせて以下を補強することです。

- 一歩方式への適合
- 守る日 / 進む日との共存
- 家族・仕事・体力・時間などの制約資源への対応
- 日次 / 週次 / 月次の編集リズムの明文化
- ホビーフォートレスへつながるための土台強化
