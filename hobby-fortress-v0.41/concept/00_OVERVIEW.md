# 全体像（00_OVERVIEW）

## このシステムが解決したいこと
- 休み時間に仕事のことを考えてしまい、趣味にエネルギーが乗らない
- 趣味を「見つける」だけでなく「育つ」状態にしたい
- 継続が苦手でも、最小行動（7/15/25分）で回路を作りたい

## 成果物（プロダクト的アウトカム）
- 趣味リスト（枝）とロードマップ（文明ツリー的）
- 趣味に紐づく「手に職つけたぜ」系の勲章（3方向の最上位）
- “機能増殖”により、やるほど「次が楽になる」演出と実用性

## コア設計
- Core Ledger（行動台帳 / 一歩ログの役割）：唯一の事実・イベントの流れ・燃料
- ノード（Spark/Tool/Workshop/Explore/Guild/Council）：行動の原子
- Armory（武器庫）：状況別に即引けるカード束（迷い削減）
- Foundry（鋳造炉）：良かった行動をカード化/強化/退役し資産化
- Synergy Map（相性マップ）：やった結果、次が楽になる“配線”
- Codex（図鑑）：巨大化しても迷子にならない参照棚（推薦しない）
- AIBridge：API無しでオンラインLLMを接続（JSONコピペ、自然文は禁止）

## ルール（憲法）
- 連鎖は1回だけ、提案は最大3、実行は最大1
- カードは最大12枚、週に新規1枚まで
- Foundry（鍛造）は週1回
- Codexは参照用（おすすめしない）
