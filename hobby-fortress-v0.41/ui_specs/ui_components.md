# UI Components（最小）

- EnergyMoodInput : energy/mood/available_min（スライダー or 3段階）
- PacketGenerator : council_request / foundry_request の生成とコピー
- PacketImporter : response貼付 → schema検証 → 反映
- CardList : 棚/時間フィルタ/カード表示
- ExecuteButton : 実行 → Ledgerへイベント追記
- LedgerView : JSONLから一覧生成
- CodexShelves : 図鑑の棚表示
